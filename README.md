# 🏎️ Mario Kart 8 Deluxe Fan Game

A browser-based kart racing game inspired by Mario Kart 8 Deluxe.

## 🎮 Features

- **42 Characters** — Full roster from Mario to Kamek
- **31 Vehicles** — Karts, bikes, and ATVs with stat differences
- **48 Tracks** — All 12 cups with 4 tracks each, procedurally generated
- **11 AI Opponents** — Rubberbanding AI with difficulty scaling
- **Item System** — Red Shell, Blue Shell, Star, Mushroom, Lightning, Banana, Bob-omb, and more
- **Grand Prix Mode** — Race through a full cup
- **Mobile Support** — Touch controls for mobile play

## 🕹️ Controls

| Action | Key |
|--------|-----|
| Accelerate | Arrow Up / W |
| Brake/Reverse | Arrow Down / S |
| Steer Left | Arrow Left / A |
| Steer Right | Arrow Right / D |
| Use Item | Space / Z / Enter |

Mobile: Touch buttons appear during races.

## 🚀 Deploy to GitHub Pages

### Method 1: Automatic (GitHub Actions)

1. Fork or push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **GitHub Actions**
4. Push to `main` branch — auto-deploys!

### Method 2: Manual

1. Go to **Settings → Pages**
2. Set Source to **Deploy from branch**
3. Select `main` branch, `/ (root)` folder
4. Your game will be live at `https://yourusername.github.io/repo-name`

## 🏁 How to Play

1. Press **START** on title screen
2. Choose **Quick Race** or **Grand Prix**
3. Select your **Character** (stats affect gameplay!)
4. Select your **Vehicle** (speed/handling tradeoffs)
5. Pick a **Track** from any of the 12 cups
6. Race 3 laps against 11 AI opponents
7. Grab **item boxes** (? boxes on track) to get items
8. Use items with **Space** to attack or boost!

## 🛠️ Technical Notes

- Pure HTML5 Canvas + Vanilla JavaScript (no dependencies)
- Procedural track generation — each of 48 tracks has a unique layout
- Themed environments with unique color palettes per track theme
- Works offline after first load
